# Complex Graphing Plotter
A Desmos-style graphing tool for **complex numbers** on the **Argand plane** (Re/Im).

## Features
- Plot vectors and loci on the complex plane
- Inequalities and regions (shaded)
- Edit, recolor, hide/show expressions
- Zoom / reset view / fit to objects
- Save & load graphs (`.IMgraph`)

## Supported input (examples)
```txt
z = 3 + 2i
z = 3pi/4
|z-(1+2i)| = 3
|z-1| < 3
2 < |z-(1+i)| < 5
arg(z-2+i) = pi
0 < arg(z) < pi/4
45deg < arg(z) < 90deg
|z-(-2+i)| = |z-(1-3i)|
